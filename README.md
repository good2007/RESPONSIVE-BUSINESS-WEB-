# Responsive Marketing Website

## Project Overview
This project is a responsive marketing website built using Tailwind CSS.

## Features
- Hero section
- Features section with 3-column responsive grid
- Pricing cards
- Footer section
- Responsive design using:
  - sm:
  - md:
  - lg:
- Hover effects
- Focus states
- Utility-first Tailwind styling

## Technologies Used
- HTML5
- Tailwind CSS (CDN)

## How to Run
1. Download the project.
2. Open `index.html` in your browser.

## Deployment

### GitHub Pages
1. Create a GitHub repository.
2. Push the project files.
3. Go to:
   Settings → Pages
4. Select:
   Branch: main
   Folder: /root
5. Save.

### Netlify
1. Login to Netlify.
2. Click "Add New Site".
3. Import from GitHub.
4. Select your repository.
5. Deploy.
